# MVP Plan

## Inputs

- free-text request,
- optional class or ascendancy,
- main skill or archetype,
- budget: starter, mid, endgame,
- goal: damage, tanky, balanced,
- playstyle: melee, projectile, spell, minion, totem, damage over time.

## Outputs

- matched archetypes,
- skill recommendations,
- support gem recommendations,
- item stat priorities,
- unique item candidates,
- unique jewel candidates,
- lineage support candidates for endgame,
- explanation for each match,
- patch impact notes when data changes.

## First Milestone

Build a tag-based recommender with small fixture data and deterministic scoring.

## Second Milestone

Add importers for real game data snapshots.

## Third Milestone

Connect candidate builds to PoB2 calculation entry points.
